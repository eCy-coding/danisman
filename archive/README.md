# archive/

Superseded and point-in-time documents. **Historical record — not source of truth.**
Kept in git for traceability; do not treat anything here as current. Live docs are
under [`docs/`](../docs/README.md).

| Folder | Contents |
| --- | --- |
| `phase-reports/` | Phase handoffs, audits, telemetry, lighthouse summaries, ship certificate, legacy deploy guide, lint triage — snapshots from specific phases (P42–P98, Track-B, etc.). |
| `prompts-raw/` | Raw request/intake dumps (`istek*`, `talep*`) and an early prompt iteration. |
| `mac/` | Personal macOS hygiene/optimization notes — not project docs. |

**Policy:** move here instead of deleting when a doc is superseded. If you need
something back as a live doc, `git mv` it into the right `docs/` bucket and update
[docs/README.md](../docs/README.md).
