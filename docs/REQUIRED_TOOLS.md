# Required System Tools & Helper Applications

To ensure the "Ecypro Premium Consulting" system operates flawlessly, the following 15 applications are required for development, debugging, asset management, and deployment.

## Development & Productivity
1.  **Top 1: Docker Desktop**
    *   **Purpose**: Containerization and consistent production environment simulation.
    *   **Link**: [Download](https://www.docker.com/products/docker-desktop)
2.  **Top 2: Postman**
    *   **Purpose**: Testing and debugging the YouTube Data API v3 and other external integrations.
    *   **Link**: [Download](https://www.postman.com/downloads/)
3.  **Top 3: Warp Terminal**
    *   **Purpose**: AI-integrated high-performance terminal for efficient command-line operations.
    *   **Link**: [Download](https://www.warp.dev/)
4.  **Top 4: Raycast**
    *   **Purpose**: System-wide productivity tool, script launcher, and window manager replacement.
    *   **Link**: [Download](https://www.raycast.com/)
5.  **Top 5: Arc Browser**
    *   **Purpose**: Modern, developer-focused browser with excellent dev-tools and workspace management.
    *   **Link**: [Download](https://arc.net/)

## Database & Backend
6.  **Top 6: TablePlus**
    *   **Purpose**: Modern GUI for relational databases (if local SQL/Postgres is added).
    *   **Link**: [Download](https://tableplus.com/)
7.  **Top 7: ngrok**
    *   **Purpose**: Exposing localhost to the internet for testing webhooks (e.g., YouTube push notifications).
    *   **Link**: [Download](https://ngrok.com/)
8.  **Top 8: Wireshark**
    *   **Purpose**: Deep network protocol analysis if low-level API connection issues arise.
    *   **Link**: [Download](https://www.wireshark.org/)

## Design & Documentation
9.  **Top 9: Figma**
    *   **Purpose**: Viewing UI/UX design specifications to ensure "Premium" aesthetics.
    *   **Link**: [Download](https://www.figma.com/)
10. **Top 10: Obsidian**
    *   **Purpose**: Knowledge base for maintaining system architecture docs and "Brain" notes.
    *   **Link**: [Download](https://obsidian.md/)
11. **Top 11: Notion**
    *   **Purpose**: Project management and collaborative documentation.
    *   **Link**: [Download](https://www.notion.so/desktop)

## Media & Assets (Critical for Video System)
12. **Top 12: VLC Media Player**
    *   **Purpose**: Reliable playback of generated video files to verify encoding integrity.
    *   **Link**: [Download](https://www.videolan.org/vlc/)
13. **Top 13: HandBrake**
    *   **Purpose**: Manual transcoding and inspection of video codecs/bitrates.
    *   **Link**: [Download](https://handbrake.fr/)
14. **Top 14: ImageOptim**
    *   **Purpose**: Lossless compression of static assets (images) for web performance.
    *   **Link**: [Download](https://imageoptim.com/mac)
15. **Top 15: GitHub Desktop**
    *   **Purpose**: Visual management of version control and large diffs.
    *   **Link**: [Download](https://desktop.github.com/)
