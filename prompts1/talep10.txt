## Deployment Seçenekleri

### Seçenek A: Vercel + Render (Önerilen)
```bash
# Frontend → Vercel (otomatik CDN, Edge Functions)
vercel --prod

# Backend → Render (managed PostgreSQL + Node.js)
render deploy

# Veritabanı
npx prisma migrate deploy
```

### Seçenek B: Docker Compose (Self-hosted)
```bash
docker compose --profile production up -d
# Frontend: nginx (port 80/443)
# Backend: Node.js (port 3001)
# Database: PostgreSQL (port 5432)
# Cache: Redis (port 6379)
```

### Seçenek C: Netlify (Sadece Frontend)
```bash
# netlify.toml zaten yapılandırılmış
netlify deploy --prod
```

## CI/CD Pipeline (.github/workflows/ci.yml)
```yaml
jobs:
  quality:
    - typecheck (frontend + server)
    - lint (ESLint)
    - test (Vitest 29/29)

  build:
    - npm run build
    - sitemap generate (41 URL)

  e2e:
    - Playwright chromium (95/95)
    - # firefox + webkit opsiyonel

  release:
    - Sentry source maps upload (tag push)
```
