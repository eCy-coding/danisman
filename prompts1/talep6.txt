- /admin/services  — Hizmet listesi yönetimi
- /admin/blog      — (route mevcut, içerik Phase 31)
- /admin/analytics — (route mevcut, içerik Phase 31)

## AdminBookingsPage Özellikleri
- Arama: ad, e-posta, şirket
- Filtre: tüm / bekliyor / onaylı / tamamlandı / iptal
- Durum badges (renk kodlu)
- Satır seçince yan panel detay açılır
- Durum güncelleme: Onayla / İptal / Tamamlandı
- Toplam / Bekleyen / Onaylı / Tamamlanan sayaçlar

## AdminSidebar Yapısı
```
Overview    → /admin/dashboard
Bookings    → /admin/bookings   ← YENİ
Services    → /admin/services
Content     → /admin/blog
Analytics   → /admin/analytics  ← YENİ
Settings    → /admin/settings
```
