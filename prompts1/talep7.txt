## Workbox Caching Stratejileri (vite.config.ts)

```javascript
runtimeCaching: [
  // API: NetworkFirst (5s timeout, 5dk cache)
  { urlPattern: /\/api\//, handler: 'NetworkFirst',
    options: { networkTimeoutSeconds: 5, maxAgeSeconds: 300 } },

  // i18n çeviri dosyaları: StaleWhileRevalidate (24sa cache)
  { urlPattern: /\/locales\//, handler: 'StaleWhileRevalidate',
    options: { maxAgeSeconds: 86400 } },

  // Font dosyaları: CacheFirst (1 yıl)
  { urlPattern: /\.(woff2|woff|ttf)$/, handler: 'CacheFirst',
    options: { maxAgeSeconds: 31536000 } },

  // Görseller: CacheFirst (30 gün, max 60 entry)
  { urlPattern: /\.(png|jpg|jpeg|webp|svg|ico)$/, handler: 'CacheFirst',
    options: { maxEntries: 60, maxAgeSeconds: 2592000 } },
]
```
