# Dijital Dönüşüm KPI Örnekleri

Dijital dönüşüm yatırımlarının %70'i hedefine ulaşamıyor. Temel neden: doğru KPI seçilmemesi veya KPI'ların stratejiyle bağlantısız izlenmesi.

Bu rehberde 4 boyutta 12 kritik KPI'ı, hesaplama formüllerini ve sektörel referans değerleri paylaşıyoruz.

## Neden Dijital Dönüşüm KPI'ları Farklıdır?

Geleneksel operasyonel KPI'lar (üretim hızı, stok devir hızı) statik sistemleri ölçer. Dijital dönüşüm KPI'ları ise:

- **Değişim hızını** (ne kadar çabuk adapte ediyoruz?)
- **Dijital olgunluğu** (teknoloji nüfuzu hangi seviyede?)
- **İnsan-sistem entegrasyonunu** (çalışanlar sistemi kullanıyor mu?)

ölçmek zorundadır.

## Boyut 1: Operasyonel Verimlilik KPI'ları

### 1. Süreç Dijitalleşme Oranı (PDR)
```
PDR = (Dijitalleşen süreç adımları / Toplam süreç adımları) × 100
```
**Hedef:** İlk 12 ayda %30 → 3 yılda %70+

### 2. Otomasyon Oranı (AR)
```
AR = (Otomatik işlem sayısı / Toplam işlem sayısı) × 100
```
**Sektör ortalaması:** Finans %45, Üretim %38, Kamu %22

### 3. Dijital Hata Oranı (DER)
```
DER = (Dijital sistemdeki hata sayısı / Toplam işlem sayısı) × 10.000
```
**Kabul edilebilir:** &lt;50 hata/10.000 işlem (ilk 6 ay), &lt;10 (12. ay)

## Boyut 2: Müşteri Deneyimi KPI'ları

### 4. Dijital NPS (Net Promoter Score)
Sadece dijital kanallar üzerinden alınan NPS; fiziksel kanaldan ayrı izlenmeli.
**Benchmark:** Fintech liderleri 65+, traditional banking 25-35

### 5. Dijital Kanal Kullanım Oranı (DCR)
```
DCR = (Dijital kanal işlem sayısı / Toplam kanal işlem sayısı) × 100
```
**2026 hedefi (Türkiye bankacılık):** %78+

### 6. Self-Servis Çözüm Oranı
```
SOR = (Self-servis çözülen talep / Toplam talep) × 100
```
Her %10 artış → müşteri hizmetleri maliyetinde ~%12 düşüş

## Boyut 3: İnovasyon Kapasitesi KPI'ları

### 7. Dijital Ürün Lansmanı Hızı (DPLV)
```
DPLV = Yılda piyasaya sürülen dijital ürün / hizmet sayısı
```
**Referans:** Dijital olgun organizasyonlar yılda 8-12 lansmanı başarıyla yönetir.

### 8. Teknoloji Borcu Oranı (TBO)
```
TBO = (Bakım/güncelleme maliyeti / Toplam IT bütçesi) × 100
```
**Sağlıklı oran:** &lt;%30. &gt;%50 → acil modernizasyon sinyali.

### 9. Veri Kalitesi Skoru (VKS)
Eksik, tutarsız, yinelenen kayıt oranı. Veritabanının **%95+** temiz olması hedeflenir.

## Boyut 4: Finansal Etki KPI'ları

### 10. Dijital Dönüşüm ROI
```
ROI = (Dijital kazanım - Yatırım maliyeti) / Yatırım maliyeti × 100
```
**Tipik ufuk:** 18-36 ay. İlk 12 ayda maliyet tasarrufu, 24. ayda gelir artışı beklenir.

### 11. IT Maliyet Verimliliği
```
IT Maliyet Verimliliği = Gelir / IT harcaması
```
**Benchmark:** Lider şirketlerde bu oran sektör ortalamasının 1.5-2×'i

### 12. Dijital Gelir Payı (DGP)
```
DGP = (Dijital kanaldan gelen gelir / Toplam gelir) × 100
```
2026 hedefi sektöre göre: E-ticaret %80+, B2B SaaS %70+, Profesyonel hizmetler %25+

## KPI Yönetim Sistemi Kurma

**Adım 1:** Strateji hizalaması — her KPI bir stratejik hedefe bağlansın (OKR formatı)  
**Adım 2:** Baseline ölçümü — dönüşüm başlamadan mevcut durum fotoğrafı  
**Adım 3:** Veri kaynağı tanımlama — her KPI için tek kaynak (single source of truth)  
**Adım 4:** Raporlama ritüeli — haftalık operasyonel + aylık stratejik review  
**Adım 5:** Aksiyon tetikleyicileri — hangi eşik değerde hangi aksiyon alınır?

## Yaygın Hatalar

- **KPI enflasyonu:** 30+ KPI izlemek odak kaybına yol açar. Maksimum 12-15 KPI.
- **Lagging-only izleme:** Sadece geçmişe bakan metriklere odaklanmak. Leading (öncü) KPI ekle.
- **İzole KPI'lar:** Birbirini etkileyen KPI'ların bağlantısını görememek.
- **Benchmark bağımlılığı:** Sektör ortalamasını geçmek yeterli değil — kendi trendine bak.
